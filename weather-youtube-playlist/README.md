# WeatherCast Playlist

현재 날씨를 기준으로 유튜브 기반 10곡 이상 재생목록을 추천하는 웹앱입니다.

## 실행

1. 터미널에서 이 폴더로 이동합니다.
2. `node server.js`를 실행합니다.
3. 브라우저에서 `http://localhost:3000`을 엽니다.

## 기능

- 지역 입력 또는 현재 위치 기반 날씨 조회
- 날씨 상태에 따른 분위기 분류
- 유튜브 검색 결과 기반 12곡 추천
- 결과 부족 시 선별곡으로 자동 보완
- 유튜브 임베드 플레이어와 목록 복사

## 참고

- 사용자가 말한 `Google 날씨`는 공식 공개 API가 없어서, 안정적으로 동작하도록 날씨 조회는 Open-Meteo를 사용했습니다.
- 추천곡은 유튜브 검색 결과를 우선 사용합니다.
